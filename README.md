# Relax masáž — landing

Статичний односторінковий лендінг для [tantraclub.cz](https://tantraclub.cz).

## Файли

- `index.html` — сторінка (CS / EN / RU, CTA WhatsApp)
- `render.yaml` — Render Blueprint (Static Site)

## Деплой

Push у `main` → автоматичний деплой на Render (сервіс `relax-massage-landing`).

## Оновлення

1. Змінити `index.html`
2. `git commit` + `git push origin main`
